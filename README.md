# 📸 AI Image Upscaler (4K Restoration)

![Banner Image](https://via.placeholder.com/800x400?text=AI+Image+Upscaler)
*(Tip: Replace this placeholder with your actual YouTube thumbnail or a before/after result image)*

This project is a 100% free, offline-capable AI Image Upscaler that utilizes **Real-ESRGAN** and **GFPGAN** to restore blurry, low-resolution photos into crisp 4K images. Instead of simply stretching the pixels, it intelligently reconstructs fine details like hair, fur, and skin textures.
